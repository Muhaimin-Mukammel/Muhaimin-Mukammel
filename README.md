### [Lightweight Database Engine](https://github.com/Muhaimin-Mukammel/Lightweight-Database-Engine)
**Main Project** — A small, single-file relational database engine written from scratch in Java.

**Key Features & Learning Highlights:**
- 📄 **Page-based storage** — Fixed 4KB pages split into 128-byte slots using `RandomAccessFile`
- 📋 **Schema Catalog** — Table definitions (columns, types, primary keys) persisted as JSON
- 🔑 **Primary Key Hash Index** — In-memory mapping of PK → (pageId, slotId), rebuilt on startup
- 📦 **Binary Row Serialization** — Custom row packing/unpacking for `INT` and `STRING` types
- 🛠️ **CRUD Service Layer** — Full create table, insert, select, delete, and getAll operations
- 🖥️ **Interactive Console Dashboard** — REPL shell for creating tables and running queries (CREATE, USE, INSERT, SELECT, DELETE)

Built as a deep learning project to understand database internals (pages, slots, indexing & persistence).
